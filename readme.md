# Ejemplos de JavaScript para el proyecto de la lista de tareas

En este repositorio encontrarás ejemplos de código JavaScript para implementar una lista de tareas (to-do list). Estos ejemplos incluyen funciones para agregar, eliminar y marcar tareas como completadas.

## Los ejemplos

Los ejemplos estarán dentro de la carpeta `ejemplos` e incluirán como máximo tres archivos:

- El código HTML para la estructura de la lista de tareas: `index.html`.
- El código JavaScript para la funcionalidad de la lista de tareas: `script.js`.
- El código CSS para el estilo de la lista de tareas: `styles.css`.

En los primeros ejemplos el código JavaScript estará incluido directamente en el archivo HTML y no se incluye nada de CSS, mientras que en los ejemplos posteriores se separará en un archivo `script.js` para una mejor organización.
